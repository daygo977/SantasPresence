Santa's Presence: A 3D Unity game in which Santa must deliver presents without getting caught (sneak/stealth).

Santa must deliver presents on Christmas day without getting caught. Santa's goal is to find the tree place the presents and escape the house as fast as posible. Santa has the ability to hide behind cover and has special abilities.
